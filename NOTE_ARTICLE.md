# サウナに行くだけでブログが自動で書かれる仕組みを作った

サウナが好きで、行くたびにブログを書きたいと思っていた。でも毎回施設を調べて、写真を整理して、経費を記録して、記事を書いて…という作業が億劫で、結局書かないことが多かった。

そこで「サウナに行って、iOSショートカットを1タップするだけで、ブログ下書きとGitHub PRが自動で作られる仕組み」を作った。

---

## 完成した流れ

```
サウナへ行く
↓
レシートをもらう
↓（ここだけ手動）
iOSショートカットを起動
  → サウナイキタイのURLを貼る
  → ボイスメモで感想を話す
  → 送信ボタンをタップ
↓（ここから全自動・約20秒）
Claude AIがWeb検索して施設情報を調査
↓
ブログ下書きをMarkdownで自動生成
↓
GitHubにブランチ作成 → PR作成
↓（月1回 自動）
Google DriveのレシートをOCRで読み取り
↓
経費をスプレッドシートに自動記録
↓（自分でやること）
PRを確認 → 感想を加筆 → マージ
↓
ブログに自動デプロイ・公開 🎉
```

---

## 技術スタック

| 役割 | 使ったもの |
|---|---|
| ブログ本体 | Astro（静的サイト生成） |
| ホスティング | Vercel |
| API処理 | Vercel Serverless Functions |
| AI記事生成 | Anthropic Claude API（Haikuモデル） |
| Web検索 | Claude の web_search ツール |
| OCR（レシート読み取り） | Google Cloud Vision API |
| レシート保管 | Google Drive |
| 経費記録 | Google Sheets |
| ソース管理・PR作成 | GitHub API |
| 月次自動実行 | GitHub Actions |
| モバイルトリガー | iOS Shortcuts |

## 工夫したポイント

### ボイスメモで感想を話すだけでOK

iOSショートカットには「音声入力」機能がある。これを使うと、話した言葉がそのままテキストに変換される。サウナから出た後にスマホに向かって「オートロウリュが最高だった、水風呂も深くて良かった」と話すだけで感想が記録される。

### Claudeが施設情報を自動で調べてくれる

サウナイキタイのURLを渡すと、Claudeが施設名・住所・料金・営業時間・サウナ室の特徴・水風呂の温度などをWeb検索で調べてMarkdownに整形してくれる。自分でやると30分かかる作業が20秒で終わる。

### 経費はまとめて月1回処理

サウナに行くたびにリアルタイムで経費処理しようとすると複雑になる。シンプルにするために「レシートはDriveに手動アップロード、月末にGitHub Actionsが一括でOCR→スプレッドシート転記」という設計にした。重複チェックも入っているので何度実行しても安全。

### 自分がやることは最小限

- レシートをDriveにアップロード
- ショートカットを1タップ
- PRで感想を加筆 → マージ

この3つだけ。

## 自分がやることになったこと

システムを作ることで、ブログを書く作業から「施設を調べる・情報を整形する・記事の構成を考える」という部分が完全になくなった。

残るのは**自分の感想を書くこと**だけ。

それが一番大事な部分なので、これはむしろ良い。

---

## まとめ

- サウナに行くたびの記録作業が1タップになった
- 経費管理も月1回のボタン1つで完結
- 自分の感想を書く時間だけに集中できるようになった

「自動化しすぎると味がなくなる」という意見もあるかもしれないが、施設情報の調査や経費記録のような繰り返し作業こそ自動化して、自分の言葉で感想を書く部分に時間を使いたい。

そういう仕組みが作れた。
